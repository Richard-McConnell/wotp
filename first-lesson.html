First lesson. HTML.

To start to write a web page you need a code editor. I'm using Sublime Text. It's cross platform. You can use for free for education purposes. Get this application and install.

Firstly, create a file "index.html". Web server is checking one of the first this filename. And if it will find a file with name "index", it will be displayed to visitor. It's working for URLs without specified filename, such as http://hol.org.uk/. In this link, you see that it has only domain name - "hol.org.uk". The main page in case of HOL will be index.php. You can try it yourself. Open the browser and type http://hol.org.uk/index.php. Enter

In our case, you need to find a class with HTML pages. Open any other class. Open any page in this class. Then go to address line and change it: delete all text after domain name and "/" [slash character]. And add there "index.html". Go.

What do you see? In an ordinal way, if you correctly typed it, you will see main page of this class. And that why your main file should have name "index". Now, let's return to our index.html.

Secondly, you need to fill this file with correctly marked text. In this class we will use HTML5 and for that language first line of your page should be <!doctype html>Enter. It says to browser in what language is written a page. Then goes tag <html>Enter. This tag can contain information about language. For example, <html lang="en-GB">. Most part of tags in the HTML language goes in pair - opening and closing tag. For this tag closing tag will be </html>.

At that moment your code should be looks like this:

							
<!doctype html>
<html>
</html>

Now, make it like this:

							
<!doctype html>
<html>
	<head>
		<!-- Change the title -->
		<title>The title of your amazing web page</title>
	</head>
	<body>
	</body>
</html>

I do not appreciate using Copy-Paste. Type it yourself.

As you see, there appeared a bit other paired tags. <head> - it's a meta part of page. Usually, in this tags you will write information about your page and where to take other code for page. You can set there charset, keywords, for search engine, tell here that you're author of this page and other staff. One of the tags of head is title tag. Where you can set title which will be appear on the tab of your browser. It's still being blank page and we going to change it. In tags <body></body> should be content which we wanna show to user. In tags <!-- --> I typed comment which tells you to change title of the page. Visitor of your page will not see comments on the page unless he will go to source code by using "Inspect" or "View Page Source". Note it and use it when you see very attractive thing and want to know how to do it or want to just get new knowledge.

If you wanna to create a page, you, probably, have an information that you wanna share. HTML have a tags for it - <p></p> - paragraph. Type or put your text inside of these tags. Then we need to add a header for our text. You can choose any header tag from <h1></h1> to <h6></h6>. Put it before the paragraph.

Your page should be something about like this:

							
<!doctype html>
<html>
	<head>
		<title>The title of your amazing web page</title>
	</head>
	<body>
		<h1>HOL.org.uk</h1>
		<p>HOL - it's a virtual school where you can learn a lot of interesting things.</p>
	</body>
</html>

From this part of lesson I will tell you about tags without HOL examples. You'll need to put them to your homework.

<hr /> - horizontal rule.

In <head> you can set charset of your page: <meta charset="utf-8">.

Images can have width and height attributes, which you can set in pixels [px], percents [%], em and others. I recommend to use the only em and in all devices, your sizes will be set properly. Usually, people are using pixels. Only, small part of them understand all wisdom of using em.

Sometimes, you need to put line break to your text. Use <br /> for it.

<pre></pre> shows preformatted text "as is". Without changes in output format.

Text selecting: bold<b>, italic<i>, strong<strong> and emphasized<em>. Always, close these tags: <em>Emphasized text</em>.

Try these pair tags yourself: <small></small>, <mark></mark>, <ins></ins>, <sub></sub>, <sup></sup>. You can all use them inside of paragraph to accent text [to make small or mark text].
