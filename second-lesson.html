Map and areas

Let's start from the map tag. That's really wonderful thing. You can create a clickable image with different links in specific areas. Look at this map from HOL. It's clickable image, your browser go wherever you click.

To make this thing you need an image, coords of clickable things and three tags: img,map and area. Put the image to HTML page like this: 

<img src="the-image.png" alt="The Image" usemap="#image-map" />

Attribute usemap defines the map name which should use this image. We need to create this map. After that map should be defined by tag map: 

<map name="image-map">
	 <area shape="rect" coords="0,0,50,50" href="#" title="Rect Link">
	 <area shape="circle" coords="100,100,25,25" href="#"  title="Circle Link">
	</map>

Code up above describes the map with square and circle links. Square will be on left top of the image with width fifty pixels:

Forms

Not many teachers are using forms in HOL classes, if to be honest I've seen no one. Programmers are using forms to gather information from visitors. You can wrap them to tables to make looks appropriate. 

<form method="POST" action="">
							  Login: <input type="text" name="login" /> <br />
							  Password: <input type="password" name="password" /> <br />
							  <input type="submit" name="submit" value="submit" />
							</form>
