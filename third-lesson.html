Third lesson. Styling.

Colours

You can set colour in HTML by three methods: name, RGB [red, green, blue] and HEX.

Colour names

Most part of colour names which you know will be right for HTML: black, white, red, orange, yellow, green, blue, indigo, violet. And there also more extravagant colours like aqua, AliceBlue, Bisque, Coral, Teal, Pink, DeepPink, HotPink. I recommend you look at names with the prefix "Light", they look good. You can see them at W3Schools.

RGB

RGB colour it's mixing of red, green and blue intensity. As you know no intensity is no light, though it's black colour. High-intensity of all channels is white. If you will mix red and blue you will get yellow. The intensity of each if these colours can be in the range from 0 to 255. And it's greater than sixteen millions colours - quite enough to make a good design. To pick a colour from RBG you can use rgb function. For example, you want to pick red with half intensity and you will need to write in property [red is first]: rgb(127, 0, 0);.

HEX

HEX and RGB are things of the same sort. HEX picks a colour from RGB palette, but you need to set it in hex numbers. There are three pairs of numbers. The first pair is red, second is green and the last one is blue - RRGGBB: 000000 - no intensity - black, FFFFFF - high-intensity - white, FF0000 - red, 00FF00 - green, 0000FF - blue, FF00FF - yellow. If all three pairs have same numbers you can make them short, 000 - black, FFF - white, 888 - grey. When you're setting colour property you need to put hashtag/number before HEX: p {color: #000;} - black colour of paragraphes, but will be better set it by this line of code: p {color: black;}. Use colour names, instead of numbers if you know name.

If you will use HEX or RGB I recommend you to find a colour picker, which will be convert chosen a colour to RGB or HEX.

Division and span tags

Before HTML5, usually, people were making a design by this tags. These are tags by which you can union your HTML code and give them style properties. By span tag [span] we are selecting/colouring characters and words. By divisions [div] - paragraphs and etc. That how will be designed pages by divisions: 

<!-- html, head and body stuff are up above -->
<div class="header">
<!-- Sign in, sign out forms and other things -->
</div>
<div class="content">
<!-- Independent content -->
</div>
<div class="footer">
Powered by <span class="my-cute-style">me</span>
</div>
<!-- closing body, html are down below -->
But it was old style and it should leave us. Now, people are using HTML5: 
<!-- html, head and body stuff are up above -->
<header>
<!-- Sign in, sign out forms and other things -->
</header>
<section>
<-- Independent content -->
</section>
<footer>
Powered by <span class="my-cute-style">me</span>
</footer>
<!-- closing body, html are down below -->

Selector

To set any property of an element of the page you need to say which one element. This part goes before curly braces and named as a selector.

selector { attributes }

In the example from colours section, you can see that in the selector is p. You can set any displayable tag in a body. You can specify selectors by different ways. div p {color: gray;} - this line of code defined that all paragraphs of all divisions should be grey. If a division has a table, wherein column of this table paragraph and this paragraph will be grey. What if you need to set paragraphs which are only in the division [not in tables, which are in the division]? In this case, you need this line of code with "greater than" symbol: div > p {color: black;}.

Class

You can define style class. For example, you need make few paragraphs, but not all, in one style. In this way, you're defining a class in style and writing in attribute class of these paragraphs name of that class. 

.one-style-paragraph {color: gray;}
<p class="one-style-paragraph">Grey paragraph.</p>
<p>This paragraph is black.</p>
<p class="one-style-paragraph">Another one grey paragraph.</p>
<p>Another black paragraph.</p>
<p>Etc.</p>
ID

For any displayable element you can set id attribute - identity. And you can use it for styling and in scripts. To specify style for element with id you should put before id name hashtag/number symbol "#". 

#one-paragraph {color: gray;}
<p id="one-paragraph">Grey paragraph.</p>
<p>This paragraph is black.</p>
<p>Another one black paragraph.</p>

Let's style it

First of all - you can style your page by three [!] ways: external file, internal style and inline style.

External file

<link type="text/css" rel="stylesheet" href="your-style-file.css">

Internal style

You can put your style in head tags: 

<style>
	body {color: blue;}
	h1 {color: orange;}
	footer p {color: gray;}
	.my-cute-style {font-size: .5em; color: pink; text-align: center;}
<style>

Inline style

Or set it in property style: 

<footer>
	<p>Powered by <span class="my-cute-style">me></p>
</footer>

Links

Links are styling in another way. Of course, you can set it by this way: a {color: orange;}. But what if you need to set style when mouse over it or user pressed link? 

a:link /* this is main style for all links */ 
{background-color: blue; color: white;}
a:visited /* this is style for visited links */ {color: gray;}
a:hover /* mouse over link */ {background-color: lightblue;}
a:active /* user pressed link */ {background-color: skyblue;}

What means code up above? It sets links with blue background and white colours. If it will be visited link it will colour it to grey. When user will move mouse over link, it will be coloured with light blue background. When user will press mouse button, background colour will be changed to skyblue.

Style properties

Before I will give you reference table I should tell you about value inherit. If you will set any style property to this value it will inherit this property from his parent tag. For example, you're setting the colour for paragraph and wrote that it has inherited colour property. If this paragraph in body, it will take colour from body, if this paragraph in column of table it will take from that column. initial is default value of property. For colour property it will be black.

You can specify image by url - url("image.gif");.
